# Gateway (Go)

Gin/Fiber-based API gateway. Responsibilities: JWT verify (Cognito), rate limit, request validation, tracing, fan-out to services.
Routes: /api/intake, /api/plan, /api/plan/:id, /api/plan/:id/replan, /api/quiz/lesson/:id, /api/progress/lesson/:id, /api/resources/search, /api/skills.
